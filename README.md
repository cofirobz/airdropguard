## What is AirdropGuard?

Most crypto airdrops are either scams or massive time-wasters.

AirdropGuard helps you:
- Get **AI + Human verified trust scores**
- See clear **risk flags** before connecting
- Use **read-only wallet intelligence** (no seed phrases ever)
- Focus only on opportunities worth your time

## Features

- Real-time AI analysis engine
- Human review layer
- Wallet safety tools
- Discord bot integration
- Supabase backend
- Deployed on Netlify

## Tech Stack

- **Frontend**: React
- **Backend**: Supabase
- **AI Layer**: Custom trust scoring
- **Deployment**: Netlify
- **Bot**: Discord integration
